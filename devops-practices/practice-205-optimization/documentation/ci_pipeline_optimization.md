# Оптимизация CI/CD пайплайна

## Базовые метрики (AS-IS)

| Job    | Время  |
| ------ | ------ |
| Lint   | 2 мин  |
| Test   | 8 мин  |
| Build  | 6 мин  |
| Deploy | 4 мин  |
| Итого  | 20 мин |

## Выявленные проблемы

1. Повторная установка зависимостей.
2. Последовательное выполнение тестов.
3. Отсутствие кэширования Docker.

## Применённые оптимизации

* Cache pip dependencies.
* Matrix тестирование.
* Docker Buildx cache.
* Разделение security-проверок.

## Результат

Общее время сократилось:

20 мин → 11 мин

Ускорение ≈ 45%.
