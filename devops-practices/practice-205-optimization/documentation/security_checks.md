# Security Checks

## SAST

Инструмент:

Bandit

Проверяет:

* небезопасные вызовы;
* использование subprocess;
* жёстко прописанные секреты;
* потенциальные инъекции.

## SCA

Инструмент:

pip-audit

Проверяет:

* CVE в зависимостях;
* устаревшие библиотеки.

## Security Gate

Пайплайн блокируется при:

* критических уязвимостях;
* ошибках SAST;
* ошибках SCA.

Результат:

Критические уязвимости не обнаружены.
