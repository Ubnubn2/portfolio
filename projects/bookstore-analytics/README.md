

# 📊 Bookstore Analytics – комплексный анализ данных книжного интернет-магазина

## ✨ О проекте

Этот проект демонстрирует **полный цикл аналитики данных** от сырых транзакций до готовых бизнес-отчётов.  
Вы увидите, как из хаоса покупок рождаются ценные инсайты: кто ваши лучшие клиенты, какие книги приносят максимум выручки, когда пик продаж и даже прогноз на следующий месяц.

Проект идеально подходит для портфолио **аналитика данных / BI‑аналитика**, так как охватывает:

- ⚙️ Проектирование базы данных и генерацию тестовых данных
- 🔍 Сложные SQL‑запросы (оконные функции, RFM‑анализ)
- 🐍 Обработку и визуализацию на Python (pandas, matplotlib, seaborn)
- 📈 Прогнозирование временных рядов (scikit‑learn)
- 📊 Интерактивный дашборд в Power BI
- 🐳 Контейнеризацию с Docker

## 🚀 Как запустить проект локально

### 1. Клонируйте репозиторий
```bash
git clone https://github.com/Ubnubn2/bookstore-analytics.git
cd bookstore-analytics
```

### 2. Установите и запустите PostgreSQL через Docker
```bash
docker run --name bookstore-db -e POSTGRES_PASSWORD=mysecretpassword -d -p 5432:5432 postgres:15
```

### 3. Создайте виртуальное окружение и установите зависимости
```bash
python -m venv venv
source venv/bin/activate      # Linux/Mac
.\venv\Scripts\activate       # Windows
pip install -r requirements.txt
```

### 4. Создайте таблицы и сгенерируйте данные
```bash
python src/create_tables.py
python src/generate_data.py
```

### 5. Запустите Jupyter Notebook
```bash
jupyter notebook
```
Откройте `stage3_analysis.ipynb` и выполните все ячейки – вы увидите все расчёты и графики.

### 6. Сгенерируйте Excel-отчёт
```bash
python src/generate_report.py
```
Файл `report_YYYYMMDD_HHMMSS.xlsx` появится в папке `reports/`.

### 7. (Опционально) Откройте дашборд Power BI
Файл `bookstore_dashboard.pbix` лежит в `reports/`. Подключитесь к вашей базе `localhost:5432` и исследуйте данные интерактивно.

## 📊 Результаты анализа

### 🔝 Топ‑10 книг по выручке
Лидером стала книга *«Удобный и пошаговый модератор»* с выручкой **39 541 руб.**.

### 📅 Динамика выручки по месяцам
Пик продаж приходится на сентябрь, спад – на февраль.

### 👥 RFM‑сегментация клиентов
Мы выделили 5 групп: **Чемпионы**, **Лояльные**, **Потенциальные**, **Спящие** и **Смешанный сегмент**. Почти 40% клиентов – смешанные, что требует дополнительного анализа.

### 📆 Заказы по дням недели
Суббота – самый активный день, четверг – самый спокойный.


### 🔮 Прогноз на следующий месяц
Линейная регрессия предсказывает выручку **~198 618 руб.**.

### 🔁 Повторные покупки
**97,5%** клиентов совершили более одного заказа, а доля выручки от повторных клиентов – **99,4%**. Это говорит о высокой лояльности аудитории.

### 🛒 Анализ корзины (часто покупаемые пары книг)
Мы выявили топ‑10 пар книг, которые чаще всего приобретают вместе. Это знание можно использовать для кросс‑продаж.

```
Топ‑10 пар книг:
---------------------------------------------------------------
1. "Удобный и пошаговый модератор" + "Оперативная служба поддержки" – 3 раза
2. "Интегрированная функциональность" + "Централизованная возможность" – 2 раза
...
```
